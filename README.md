# LcookIn

Website resep makanan dari Indonesia dan seluruh dunia.

## Fitur

- **Login / Daftar** – User harus login atau daftar sebelum masuk ke website.
- **Navbar** – Home, About, Contact, dan tombol **+ Tambah Resep**.
- **Kolom search** – Di bawah navbar untuk cari resep (nama, bahan, negara).
- **Home**
  - **Cari Resep dengan AI** – Ketik makanan/bahan/negara, pilih porsi, dapat resep lengkap + cara masak.
  - **Top 10 Makanan Indonesia** – Resep lengkap dan cara masak.
  - **Top 10 Makanan Dunia** – Resep lengkap dan cara masak.
- **About** – Tentang LcookIn.
- **Contact** – Form hubungi kami.
- **Tambah Resep** – User bisa menambah resep sendiri (nama, daerah, porsi, waktu, bahan, cara masak).

## Menjalankan

```bash
cd LcookIn
npm install
npm run dev
```

Buka http://localhost:5173. Jika belum punya akun, klik **Daftar** dulu, lalu login.

## Build

```bash
npm run build
npm run preview
```
